# Сайт-визитка ООО «МК ПРОФИ ЭНЕРГО ГРУПП» (ООО «МК ЭНЕРГО»)

Одностраничный лендинг в стиле uneco.ru: корпоративный стиль, синие акценты (#003087), адаптивная вёрстка.

## Структура

- `index.html` — главная страница (все блоки на одной странице, якорные ссылки)
- `privacy.html` — политика конфиденциальности
- `css/style.css` — стили
- `js/main.js` — слайдер hero, бургер-меню, плавный скролл, анимации при скролле, форма
- `images/` — папка для изображений (см. `images/README.txt`)

## Что нужно сделать перед публикацией

1. **Изображения** — положите в `images/` файлы: `slide1.jpg`–`slide4.jpg` (слайдер), `project1.jpg`–`project4.jpg` (проекты). До этого hero покажет градиент, проекты — плейсхолдеры.
2. **Контакты** — замените в `index.html` телефон `8 (495) 123-45-67`, email `info@mk-energo.ru`, ссылки на WhatsApp/Telegram и при необходимости адрес.
3. **Форма** — сейчас форма выводит сообщение и сбрасывается. Для отправки на почту нужен бэкенд (PHP, Formspree, Telegram Bot и т.п.) или сервис форм.
4. **Карта** — в блоке контактов замените плейсхолдер на iframe с Яндекс.Картами или Google Maps.
5. **Цифры в блоке «О компании»** — при необходимости измените «9+ лет», «300+ объектов» и т.д. на актуальные.

## Запуск

Откройте `index.html` в браузере или поднимите локальный сервер (например, `npx serve .` в папке проекта).

## Технологии

HTML5, CSS3 (переменные, Grid, Flexbox), ванильный JavaScript. Шрифты: Montserrat, Open Sans (Google Fonts).
